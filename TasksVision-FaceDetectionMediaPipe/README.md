# MediaPipe Face Detection

> Face detection with MediaPipe.

## Overview

MediaPipe Face Detection is a **Android learning sample** maintained by [Sohaib Ahmed](https://github.com/itssohaibahmed).

## Tech stack

- **Kotlin**
- **Android Jetpack**
- **Gradle**

## Getting started

### Prerequisites

- Android Studio (latest stable)
- JDK 17+
- Android SDK

### Run locally

1. Clone the parent repository.
2. Open `TasksVision-FaceDetectionMediaPipe` in Android Studio.
3. Sync Gradle and run the `app` module.
